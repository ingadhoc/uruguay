=========================
Uruguay Chart of Accounts
=========================

This module has configured enviroment for Uruguay's companies:

* Chart of Accounts

Configuration
=============

* Install this module.
* Got to companies and create a new one
* Got to General Settings / Accounting / Set Fiscal Chart of Account for Uruguay
